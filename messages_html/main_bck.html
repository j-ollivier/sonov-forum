Principes généraux

Ce site regroupe des aventures jouées par des tablées de joueurs dans la tradition du jeu de rôles. Tout se fait par écrit : par message sur le site ou par mail. Chaque aventure est unique, parfois dans des mondes très différents. Chaque membre inscrit peut incarner un personnage par aventure s'il y a de la place (allez si vous mourrez vous pourrez jouer la suite avec un autre personnage :P). Ce personnage peut être un maître du jeu ou un simple aventurier. Les MJ possèdent des outils supplémentaires et posent le rythme de la partie. Les joueurs réagissent aux situations décrites par le MJ et peuvent chatter ensemble.

  • Un membre peut avoir un ou plusieurs personnages. Faites bien la différence ! Un personnage peut être un maître de jeu ou un personnage joueur. Vous pouvez donc être inscrit et participer à plusieurs aventures : soit en tant que MJ, soit en tant que personnage joueur.
  • Une aventure : un MJ raconte l'histoire à plusieurs personnages. Le MJ vous décrit ce qui se passe, vous délibérez entre joueurs et vous agissez.
  • Chaque aventure possède ses règles et son rythme : tout est précisé sur la page d'accueil de chaque aventure.
  • Toutes les aventures sont publiques, pas besoin de faire partie de l'aventure ni être inscrit pour lire ce qui s'y passse.
  • Le chat est pour la discussion informelle, le message qui décrit ce que vous allez faire est (musique dramatique) très sérieux et non modifiable.
  • Le chat n'est pas mis à jour automatiquement (pour épargner le matériel qui date). Donc rafraichissez la page ou naviguez sur le site pour avoir les mises à jour. Si vous trouvez que c'est pourri, désolé !
  • Si vous voulez lancer votre propre aventure, envoyez un mail à voxclam.indes[at]gmail.com :)

Une aventure c'est...

Le principe d'une aventure est le suivant : après avoir créé votre personnage en accord avec le MJ, vous attendez patiemment que la partie commence. Elle débute avec un premier tour posté par le MJ sur le site (accessible dans la partie 'table de jeu'). Le tour à une durée limitée, quelques jours en général. En lisant le texte du MJ, vous aurez envie de décrire votre action ou chatter de manière informelle avec les autres personnages. Il est possible de chatter en public ou d'envoyer des messages privés. Le MJ laissera aussi des renseignements importants dans le lexique du monde (lieux, personnages, créatures...)

Votre action est très importante, il faut la décrire convenablement et surtout être sûr de vous : pas de modification possible ! Une fois que vous avez posté, il ne vous reste plus qu'à attendre le tour prochain que le Mj concoctera.

Chaque aventure est publique, il n'y a donc pas besoin d'être membre pour en être spectateur.

Les rubriques de chaque aventure

  • L'accueil est là pour vous pitcher l'aventure, le système adopté, les règles spécifiques et le guide de préparation du personnage.
  • La fiche de personnage permet de voir votre historique, vos caractéristiques, compétences, possessions.
  • Le monde et ses catégories permettent de voir les éléments de l'aventure que le MK vous a préparés.
  • Le menu tours de jeu permet de retrouver rapidement une partie de l'aventure.

Le coin des MJ

Le site fait bien la différence entre les personnages que vous incarnez. Lorsque vous participez à une aventure en tant que MJ, vous aurez un paquet d'options supplémentaires qui seront débloquées. A vous d'en faire bon usage.

  • Tous les messages MJ acceptent le HTML. Affichage de médias (audio/vidéo/image), hyperliens, mise en forme du texte, tout est possible.
  • Si vous voulez coder et inscrire vos stats dans la DB plutôt qu'en texte libre, il faut m'envoyer un mail et on fait ça :)
  • Accès à votre aventure via l'interface d'administration.
  • Possibilité de modifier et supprimer les messages/entités en quelques clics
  • Changer la visibilité de chaque entité de jeu (lieu, PJ, PNJ)
  • Envoyer un mail en un clic à chaque personnage participant pour les prévenir du nouveau tour.

Le mieux ça reste de passer un petit moment avec moi pour voir tout ce qui est possible :)

Sous le capot

Le site de l'Ogre est installé sur un vieux laptop Asus dans mon salon, et commme je le disais sur mon autre site sonov.fr, il fait du bruit mais j'aime bricoler dessus. Le site utilise Django 1.10, sur lequel je fais passer Gunicorn en websocket pour qu'il discute avec Nginx, le server frontend. La donnée est stockée sur mon serveur postgres et les tables sont générées via les modèles django. Chaque aventure vient avec un ensemble de modèles basique. C'est du fait maison et en mode didactique donc oui il y aura des ratés. Le serv ferme ses portes régulièrement entre 3h et 3h10 du matin pour maintenance. Si vous avez des connaissances poussées, je suis toujours à l'écoute pour échanger sur le sujet.

Pour voir le code du site (sauf les identifiants secrets, petit malin) il faut aller sur mon repo github : ICI. Vous pourrez aussi voir les autres projets que j'ai menés sans prétention ICI.
